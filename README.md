# React-Setting
React 프로젝트 초기 구성을 잡기 위한 프로젝트

## Info
- - -
### Version
- react: 18.2.0
- Node.js: 18.15.0
- create-react-app: 5.0.1

### Installation
```shell
npm install

# 기설치된 패키지 및 라이브러리
npx create-react-app react-setting
yarn add immer
yarn add @sentry/browser
```

### Start
```shell
npm start
```

### Build
```shell
npx serve ./build
```
